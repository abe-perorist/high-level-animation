# High-Level Animation Demo

Scroll-driven concept site inspired by DP Studio, built with Next.js (App Router), TypeScript, Tailwind CSS, and GSAP ScrollTrigger.

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to explore the prototype.

## Deploying

This project is ready for Vercel. Push to a Git repository and import it into Vercel. No additional configuration is required.

## Key Features

- Hero, Content, and Footer sections styled with Tailwind CSS.
- ScrollTrigger-powered pin and scale effect for the Content section heading.
- Parallax-inspired horizontal motion for three feature cards during the remainder of the Content section scroll.
